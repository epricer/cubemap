# README #

### What is this repository for? ###

* Shows a map of an office.

### How do I get set up? ###

* fill out the map.json file (it will auto-scale to fit, so the units are arbitrary)
* install go
* `$go run cubemap.go -port [desired port]`'
* visit http://servername:port/
